About this app

Usage:

  • Use the 'n' key to move next, 'p' key for previous, and 'enter' to select

Methodology:

Five equations were entered on the calculator with either a three key or five key switch interface. The three key interface used 'n', 'p', and 'enter', for next, previous, and select respectively. The five key interface added 'u' and 'd' for up and down, respectively.

The five equations were:

  1. 5+10*3
  2. 18/3-10
  3. 4/3/2*3
  4. 71+549/17
  5. 852*7+96-74

The test included four trials for each interface. Completion times were averaged and are listed in the table below.

Test results:

Average Time for Task Completion

Switch Interface Average Time to Completion Number of Errors
Three Key 98.43 seconds 2
Five Key 110.44 seconds 0

These results indicate that the simpler three-key interface is slightly faster, but did produce more errors. This increase in speed could be explained by the mapping of the keys to switches, with the 'u' and 'd' keys in the five key interface being quite distant from eachother.

A different key mapping would likely yield different results than were collected here.
